# Texturas

Coloque seus arquivos de textura aqui.

## Formatos Suportados
- `.png` - Imagens PNG
- `.jpg` / `.jpeg` - Imagens JPEG
- `.hdr` - High Dynamic Range (para iluminação baseada em imagem)

## Como Usar
1. Adicione seus arquivos de textura nesta pasta
2. Acesse o viewer e selecione a textura desejada na seção "Texturas Disponíveis"
3. A textura será aplicada como fundo da cena 3D
